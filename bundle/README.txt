Ca2+tchAll -- Portable Windows Bundle
======================================

No Python installation required.

FIRST TIME
----------
1. Double-click setup.bat
   - Downloads embedded Python 3.12 (~10 MB)
   - Installs dependencies (~50 MB)
   - Needs internet access, runs once only

EVERY TIME AFTER
----------------
2. Double-click launch.bat
   - Starts the app and opens your browser at http://localhost:8002
   - Close the window to stop the app

REQUIREMENTS
------------
- Windows 10 or later (64-bit)
- Internet access for first-time setup only
